# CCS 610 - Cloud Computing Concepts

This directory contains coursework from CCS 610 - Cloud Computing Concepts, an introductory cloud computing course in the M.S. Cloud Computing Architecture program at UMGC. The projects center around BallotOnline, a fictional e-voting company undergoing a cloud migration analysis, covering strategy development, risk and compliance assessment, and cloud provider evaluation.

* [Project 1: Cloud Migration Strategy for BallotOnline](project-1-cloud-migration-strategy/)
* [Project 2: Risk Assessment and Compliance](project-2-risk-and-compliance/)
* [Project 3: AWS vs Azure Provider Evaluation](project-3-aws-azure-evaluation/)
