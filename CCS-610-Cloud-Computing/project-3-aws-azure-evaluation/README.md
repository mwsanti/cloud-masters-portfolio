# Project 3: AWS vs Azure Provider Evaluation

## Objective:
Evaluated Amazon Web Services (AWS) and Microsoft Azure as cloud providers for BallotOnline's migration. This project involved analyzing cloud architectures, assessing migration trends and feasibility for each provider, and producing a final comparative evaluation report to recommend the optimal platform for BallotOnline's e-voting infrastructure.

## Technologies Used:
* Amazon Web Services (AWS)
* Microsoft Azure
* Cloud Architecture Design
* Microsoft Office (Word)

## Key Concepts Demonstrated:
* **Cloud Provider Comparison:** Conducted a side-by-side evaluation of AWS and Azure across service offerings, pricing, reliability, and suitability for e-voting workloads.
* **Cloud Architecture Analysis:** Analyzed and documented cloud architecture patterns applicable to BallotOnline's infrastructure requirements.
* **Migration Feasibility:** Assessed the feasibility of migrating BallotOnline to each provider, considering technical complexity, cost, and organizational readiness.
* **Industry Trends:** Researched current cloud migration trends and their implications for provider selection and migration planning.
* **Architecture Diagramming:** Created architectural diagrams illustrating BallotOnline's proposed cloud deployment.

## Implementation Steps (High-Level):
1. **AWS Feasibility Analysis:** Evaluated AWS service offerings and assessed feasibility for hosting BallotOnline's e-voting platform.
2. **Azure Analysis:** Conducted a parallel evaluation of Microsoft Azure's capabilities and suitability for the migration.
3. **Cloud Architecture Design:** Analyzed and documented cloud architecture patterns for BallotOnline's deployment.
4. **Trends Research:** Investigated current cloud migration trends and industry shifts affecting provider selection.
5. **Architecture Diagram:** Created a diagram report illustrating BallotOnline's proposed cloud infrastructure.
6. **Final Evaluation Report:** Produced a comprehensive comparison report evaluating AWS and Azure with a final provider recommendation.
