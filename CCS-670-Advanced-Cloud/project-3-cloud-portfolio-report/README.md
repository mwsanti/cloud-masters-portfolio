# Project 3 - Cloud Portfolio and Cost Analysis

## Objective:
Produce a comprehensive business analysis and capstone deliverable for the BallotOnline cloud journey, including a cloud services offerings evaluation, monthly cost analysis, total cost of ownership (TCO) analysis, return on investment (ROI) analysis, a full cloud portfolio report, and an executive presentation summarizing all findings.

## Technologies Used:
* AWS Pricing Calculator
* Azure Pricing Calculator
* TCO Analysis Frameworks
* ROI Analysis Frameworks
* Microsoft PowerPoint

## Key Concepts Demonstrated:
* **Cloud Services Evaluation:** Assessed and compared cloud service offerings across providers to determine the optimal platform mix for BallotOnline.
* **Monthly Cost Analysis:** Calculated projected monthly cloud expenditures to establish an operational cost baseline for the migrated environment.
* **Total Cost of Ownership (TCO):** Performed a full TCO analysis comparing on-premises infrastructure costs against cloud deployment costs over a defined time horizon.
* **Return on Investment (ROI):** Quantified the expected financial return from the cloud migration to justify the investment to stakeholders.
* **Executive Communication:** Synthesized all technical and financial findings into a cloud portfolio report and executive presentation suitable for senior leadership decision-making.

## Implementation Steps (High-Level):
1. Evaluated cloud service offerings and documented platform capabilities relevant to BallotOnline requirements.
2. Conducted a monthly cost analysis projecting recurring cloud expenses for the target architecture.
3. Performed a TCO analysis comparing on-premises and cloud total costs over the planning period.
4. Completed an ROI analysis quantifying the financial benefits of the cloud migration.
5. Compiled all analyses into a comprehensive cloud portfolio report.
6. Created an executive presentation summarizing the BallotOnline cloud journey, technical findings, and business justification.
