# CCS 670 - Advanced Cloud Services

## Course Overview

This capstone-level course covers advanced cloud services across both Azure and AWS platforms. Coursework includes deploying proof-of-concept environments in Microsoft Azure, leveraging advanced AWS features such as High Performance Computing (HPC), Amazon Redshift, and AWS Service Catalog, implementing data protection strategies, and producing comprehensive business cost analyses. The course culminates in a cloud portfolio report and executive presentation summarizing the full BallotOnline cloud journey.

## Projects

| # | Project | Description |
|---|---------|-------------|
| 1 | [Azure Proof of Concept](./project-1-azure-poc/) | Deployed BallotOnline PoC in Microsoft Azure with web server VMs, VM Scale Sets, Load Balancer, and PaaS Web Application |
| 2 | [Cloud Automation and Advanced Features](./project-2-cloud-automation/) | Hands-on AWS HPC (ParallelCluster), Amazon Redshift, S3 ransomware protection, AWS Service Catalog, and advanced data protection |
| 3 | [Cloud Portfolio and Cost Analysis](./project-3-cloud-portfolio-report/) | Comprehensive business analysis including TCO, ROI, monthly cost analysis, cloud portfolio report, and executive presentation |

## Technologies Used

* Microsoft Azure (VMs, VM Scale Sets, Load Balancer, PaaS Web Apps, ARM Templates)
* AWS ParallelCluster (HPC6a instances)
* Amazon Redshift
* AWS Service Catalog
* AWS CloudFormation
* Amazon S3 (Bucket Policies, Ransomware Protection)
