# Cloud Computing Masters Portfolio

Welcome to my Cloud Computing Masters Portfolio! This repository showcases coursework, lab exercises, and projects completed during my Master of Science in Cloud Computing Architecture at the University of Maryland Global Campus (UMGC). It serves as a comprehensive collection of hands-on experience across cloud platforms, infrastructure automation, governance, migration, and security.

## Key Skills Demonstrated

* **Cloud Platforms:** Amazon Web Services (AWS), Microsoft Azure, Google Cloud Platform (GCP)
* **Infrastructure as Code (IaC):** Terraform, AWS CloudFormation, Azure Resource Manager (ARM) Templates
* **Networking:** VPC/VNet design, Subnets, Internet Gateways, Load Balancers, Security Groups
* **Compute & Containers:** EC2, ECS, Azure VM Scale Sets, Azure App Services
* **Storage & Databases:** S3, RDS, Azure Blob Storage, Amazon Redshift
* **Security & Compliance:** IAM, Risk Assessment, NIST Frameworks, Data Protection, Ransomware Defense
* **Automation & Scripting:** AWS CLI, Python (Boto3), Bash, YAML/CloudFormation
* **Migration:** AWS Application Discovery Service, Data Migration, Post-Migration Monitoring
* **Monitoring:** CloudWatch, Azure Monitor, Health Checks, Dashboards
* **Governance:** Cloud Adoption Policies, SLA Management, Vendor Governance, Service Catalogs
