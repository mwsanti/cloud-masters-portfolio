# CCS 625 - Cloud Networking and Infrastructure

## Course Overview

Advanced networking course covering network protocol analysis, AWS cloud infrastructure provisioning, and cloud migration strategies. Coursework progressed from foundational networking concepts through hands-on AWS CLI operations to a full cloud migration lab environment with load balancing and monitoring.

## Projects

| # | Project | Description |
|---|---------|-------------|
| 1 | [Networking Fundamentals](./project-1-networking-fundamentals/) | Wireshark packet analysis, EC2 instance creation, subnetting exercises, and SDN discussion |
| 2 | [AWS CLI](./project-2-aws-cli/) | VPC/Internet Gateway creation via AWS CLI, CloudFormation deployment, and Boto3 EC2 provisioning |
| 3 | [Cloud Migration Lab](./project-3-cloud-migration-lab/) | AWS migration environment with VPC, load balancers, web application deployment, and monitoring |

## Key Skills Developed

* Network protocol analysis using Wireshark (DNS, HTTP, TCP/IP)
* AWS infrastructure provisioning via CLI and CloudFormation
* Python automation with Boto3 for EC2 management
* Cloud migration planning and execution
* Load balancer configuration and health check monitoring
* Subnetting and network architecture design
