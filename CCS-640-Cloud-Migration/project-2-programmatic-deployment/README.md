# Project 2 - Programmatic Cloud Deployment

## Objective:
Evaluate and compare programmatic cloud deployment tools, discuss cloud migration programmatic approaches, and produce a final programmatic deployment strategy document for the BallotOnline migration.

## Technologies Used:
* AWS CloudFormation
* Azure Resource Manager (ARM) Templates
* Terraform (HashiCorp)

## Key Concepts Demonstrated:
* **Infrastructure as Code (IaC) Comparison:** Compared CloudFormation, ARM Templates, and Terraform across capabilities, cloud provider support, template syntax, and ecosystem maturity.
* **ARM Template Deployment:** Discussed and demonstrated Azure Resource Manager template deployments as a programmatic approach to cloud infrastructure provisioning.
* **Programmatic Migration Approaches:** Evaluated different programmatic strategies for automating cloud migration tasks including resource provisioning, configuration management, and deployment orchestration.
* **Deployment Automation:** Documented the end-to-end programmatic deployment process with 16 screenshots capturing each stage of the workflow.

## Implementation Steps (High-Level):
1. Researched and compared programmatic cloud deployment tools (CloudFormation, ARM Templates, Terraform).
2. Evaluated each tool's strengths and limitations for the BallotOnline migration use case.
3. Discussed cloud migration programmatic approaches and their applicability to the project.
4. Performed ARM template deployments and documented the process.
5. Produced the final programmatic approach document recommending the optimal deployment strategy.
