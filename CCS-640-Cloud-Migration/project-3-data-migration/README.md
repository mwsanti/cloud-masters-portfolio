# Project 3 - Data Migration

## Objective:
Compare cloud data migration tool sets, evaluate data migration versus application migration strategies, and produce a comprehensive cloud migration technical report for the BallotOnline migration.

## Technologies Used:
* AWS Database Migration Service (DMS)
* AWS DataSync
* AWS Snow Family
* Cloud-native data migration tools

## Key Concepts Demonstrated:
* **Data Migration Tool Comparison:** Evaluated and compared multiple cloud data migration tool sets to determine the best approach for moving BallotOnline's data to the cloud.
* **Data vs. Application Migration:** Analyzed the differences between data migration and application migration strategies, including when each approach is appropriate and how they complement each other.
* **Migration Execution Documentation:** Captured Step 5 and Step 6 of the migration process with 13 screenshots demonstrating the hands-on data migration workflow.
* **Technical Reporting:** Synthesized tool comparisons, strategy evaluations, and migration execution findings into a final cloud migration technical report.

## Implementation Steps (High-Level):
1. Researched and compared cloud data migration tool sets across features, performance, and suitability.
2. Evaluated data migration strategies against application migration strategies for the BallotOnline use case.
3. Executed Step 5 of the data migration process (7 screenshots documenting the workflow).
4. Executed Step 6 of the data migration process (6 screenshots documenting the workflow).
5. Produced the final cloud migration technical report consolidating all findings.
