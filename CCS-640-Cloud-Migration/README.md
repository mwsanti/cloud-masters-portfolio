# CCS 640 - Cloud Migration

## Course Overview

This course covers the end-to-end cloud migration lifecycle, from initial pre-migration assessment through post-migration monitoring and management. Using BallotOnline as the target application, the coursework walks through each phase of a real-world cloud migration: discovering and evaluating on-premises infrastructure, selecting and executing programmatic deployment strategies, migrating data to the cloud, and establishing post-migration monitoring with cutover and rollback plans.

## Projects

| # | Project | Description |
|---|---------|-------------|
| 1 | [Pre-Migration Assessment](./project-1-pre-migration-assessment/) | Infrastructure discovery using AWS Application Discovery Service, tool comparisons, and final migration planning |
| 2 | [Programmatic Cloud Deployment](./project-2-programmatic-deployment/) | Evaluation of programmatic deployment tools (CloudFormation, ARM Templates, Terraform) and ARM template deployments |
| 3 | [Data Migration](./project-3-data-migration/) | Cloud data migration tool set comparison, data vs. application migration strategy evaluation, and technical report |
| 4 | [Post-Migration](./project-4-post-migration/) | AWS monitoring/management evaluation (CloudWatch, CloudFormation), cutover/rollback options, and post-migration activity reporting |

## Technologies Used

* AWS Application Discovery Service
* AWS CloudFormation
* AWS CloudWatch
* Azure Resource Manager (ARM) Templates
* Terraform
* AWS Migration Tools
