# CCS 630 - Cloud Infrastructure Operations and Governance

This directory contains coursework from CCS 630 - Cloud Infrastructure Operations and Governance, a course in the M.S. Cloud Computing Architecture program at UMGC. The projects center around BallotOnline, covering cloud adoption policy development, Terraform infrastructure deployment, GCP workload evaluation, VPC and S3 deployment for backups and archiving, and AWS Service Catalog operations.
