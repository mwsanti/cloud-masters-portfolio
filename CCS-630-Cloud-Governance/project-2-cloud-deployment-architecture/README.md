# Project 2: Cloud Deployment Architecture

## Objective:
Evaluated available cloud service offerings for BallotOnline including backup solutions and software development environments, created user stories to capture functional requirements, monitored cloud resource usage, conducted a Google Cloud Platform hands-on lab, and produced a final cloud deployment architecture plan.

## Technologies Used:
* Amazon Web Services (AWS)
* Google Cloud Platform (GCP)
* Microsoft Office (Word, Excel)

## Key Concepts Demonstrated:
* **Cloud Service Offering Evaluation:** Assessed cloud service offerings for backup and software development use cases, comparing features, pricing, and suitability for BallotOnline's workloads.
* **User Story Development:** Created user stories in spreadsheet format to define functional requirements and acceptance criteria for cloud deployment.
* **Cloud Resource Monitoring:** Monitored cloud resource usage to evaluate performance, utilization, and cost efficiency of deployed services.
* **Multi-Cloud Exploration:** Conducted a Google Cloud Platform lab to gain hands-on experience with GCP services and compare capabilities across cloud providers.
* **Deployment Architecture Planning:** Produced a final cloud deployment architecture plan defining the target infrastructure, service selection, and deployment strategy for BallotOnline.

## Implementation Steps (High-Level):
1. Evaluated available cloud service offerings, analyzing options for backup and software development workloads.
2. Assessed cloud service offerings specifically for backup solutions, comparing storage tiers, durability, and recovery options.
3. Assessed cloud service offerings for software development environments, evaluating CI/CD, compute, and tooling options.
4. Developed user stories in Excel to capture stakeholder requirements and define acceptance criteria for deployment.
5. Monitored cloud resource usage across deployed services to track utilization, performance metrics, and cost.
6. Completed a Google Cloud Platform lab to gain practical experience with GCP infrastructure and services.
7. Compiled findings into a final cloud deployment architecture plan outlining the recommended infrastructure and deployment approach.
